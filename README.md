# 搖滾寶寶 Rock Bae

## 簡介：

可使用此網站交流台灣獨立樂團心得、並得知最新的表演資訊。（目前已停止更新）

## 作者：

[林思彤](https://github.com/cianyastone)、[方 雁](https://github.com/fyan0125)

## 技術：

1. 前端框架：React.js
2. 資料庫：Firebase

## 功能：

1. 查看音樂季資訊（包括演出陣容和主辦單位）、並於網頁中購票。
2. 註冊後可於論壇中查看、發布文章，並留言、收藏。
